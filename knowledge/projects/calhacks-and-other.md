## The Project: Mind-Controlled Music (Duet)
**Devpost: devpost.com/software/duet-0tbkxe**
**Stack**: Python, Gemini, Sonic Pi

**What we built**: A system that translates real-time EEG brainwave data into live music composition. You think — the music changes. Built with full hardware integration, real-time inference, and audio generation at sub-200ms latency.

**Technical details**:
- Deployed full ML pipeline: EEG preprocessing → emotion classification → LLM-driven composition
- Used random forest classifier (scikit-learn) and Gemini API for real-time emotion-to-music translation
- Sub-200ms latency from brainwave signal to audio output — fast enough to feel responsive

**Recognition**: 1st Overall out of 2,500+ teams. World's largest student hackathon.

**What it demonstrates**: I can build technically sophisticated, novel systems under extreme time pressure (36-48 hour build window). The project combined hardware, ML, and LLM APIs in a way that required all three to work simultaneously — no hiding a non-functional demo. Also demonstrates breadth: health AI by day, brain-computer interfaces at hackathons.
